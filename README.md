# AWS re/Start – My 12-Week Cloud Learning Journey 

## Overview

This repository documents my learning journey through the **AWS re/Start program**, a 12-week intensive training designed to build foundational and practical skills in **cloud computing and AWS technologies**.

During the program I explored how modern cloud environments work, gained hands-on experience with core AWS services, and practiced solving real-world scenarios that cloud professionals encounter in their daily work.

The purpose of this repository is to **document what I learned, showcase practical labs and exercises, and demonstrate my growing cloud skillset.**

---

## What I Learned

Over the course of the program, I developed knowledge and hands-on experience in several key areas of cloud computing:

* Cloud computing fundamentals and AWS global infrastructure
* Core AWS services and how they interact
* Linux basics and working in the command line
* Networking concepts such as IP addressing, routing, and DNS
* Security principles and identity management in the cloud
* Databases and different storage solutions
* Monitoring, troubleshooting, and system reliability
* Basic cloud architecture and best practices

Throughout the training, I continuously practiced **building, configuring, and securing cloud resources**, which helped me understand how cloud environments operate in real-world scenarios.

---

## AWS Services Explored

During the program I worked with several AWS services, including:

* **Amazon EC2** – cloud-based virtual servers
* **Amazon S3** – scalable object storage
* **AWS IAM** – identity and access management
* **Amazon VPC** – networking in the cloud
* **Amazon RDS** – managed relational databases
* **AWS Lambda** – serverless computing
* **Amazon CloudWatch** – monitoring and logging
* **AWS CloudFormation** – infrastructure as code
* **Elastic Load Balancing**
* **Amazon Route 53**

---

## Hands-On Practice

A key part of the program was learning by doing. I worked on multiple practical labs where I practiced:

* Launching and managing EC2 instances
* Hosting content in Amazon S3
* Configuring secure access with IAM roles and policies
* Setting up cloud networking environments
* Monitoring system performance and logs
* Understanding scalable and reliable cloud architectures

These exercises helped transform theoretical concepts into **practical, real-world skills.**
