In this lab, I explored the basic functionality of Amazon EC2, AWS’s scalable compute service that allows users to run virtual servers in the cloud. I launched an EC2 instance configured as a simple web server and practiced managing it through the AWS console.

During the exercise, I modified the security group to allow HTTP access, monitored the instance, and resized it to understand how compute capacity can be scaled based on workload requirements. I also tested termination protection, which prevents accidental deletion of important resources.

This lab helped me understand the full lifecycle of an EC2 instance—from launching and configuring it to scaling and safely terminating the resource.
